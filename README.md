<h1 align="center">📄 Texify</h1>

<p align="center">
A simple <b>Django-based Text Analyzer Web Application</b> that helps users clean and transform text quickly.
</p>

<p align="center">
<a href="https://texify-34yv.onrender.com"><b>🌐 Live Demo</b></a> |
<a href="https://github.com/SUMAN1040/Texify"><b>📦 GitHub Repo</b></a>
</p>

<hr>

<h2>✨ Features</h2>

<ul>
<li>🔤 Remove punctuations</li>
<li>🔠 Convert text to uppercase</li>
<li>📄 Remove new lines</li>
<li>🧹 Remove extra spaces</li>
<li>🔢 Character counter</li>
<li>💻 Clean Bootstrap UI</li>
</ul>

<hr>

<h2>🛠 Tech Stack</h2>

<ul>
<li>Python</li>
<li>Django</li>
<li>HTML</li>
<li>CSS / Bootstrap</li>
<li>JavaScript</li>
<li>Render (Deployment)</li>
</ul>

<hr>

<h2>⚙️ Run Locally</h2>

<pre>
git clone https://github.com/SUMAN1040/Texify.git
cd Texify
pip install -r requirements.txt
python manage.py runserver
</pre>

<p>Open in browser:</p>

<pre>
http://127.0.0.1:8000
</pre>
